Question 2: What are the total number of accidents for each month for the various years?

Plot:


Color Scheme Choice: We choose this shade of blue for the bars because it is colorblind friendly.

Marks: The marks for this plot are the points for the values as well as the line connection marks that connect the point marks.

Channels: The channels are horizontal and vertical position for each value attribute and aligned lengths to express these quantative values. An identity channel used is the horizontal spatial position channel of the points to correspond to the ordering of the labels (ordered) for the key (month) while also seperated horizontally (separated and ordered by key attribute into horizontal regions).



Question 3: What are the total number of accidents for each year, month, and hour of day?

Plot:


Color Scheme Choice: We choose this shade of orange for the bars because it is colorblind friendly.

Marks: The marks for this plot are the lines for vertical positions that correspond to the bars representing each year/month/hour of day (geometric primitives for key items).

Channels: The channels in this plot include the length of the bar (length of marks) to express quantative values (magnitude) for the number of accidents (expresses value attribute with aligned vertical position). Also an identity channel used is for horizontal spatial position channel of the bars (marks) to correspond to the ordering of the labels (ordered) for the key (month/year/hour of day), thus seperated horizontally while also aligned vertically.
